# MovingBox
Small react component for 0 config F.L.I.P. animations


## Examples

From this
```jsx
<div> this thing will never be animated </div>
```

To this
```jsx
<MovingBox> this thing will be nicely animated on any style change</MovingBox>
`````


## Props

from?: From;

fromFade?: boolean;

debugName?: string;

animKey?: string;

as?: React.ElementType;
